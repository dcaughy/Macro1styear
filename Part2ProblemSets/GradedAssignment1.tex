First, we can simplify the above funtion to 
\[f(k)(\frac{m(\lambda)}{\lambda\delta+m(\lambda)})-k\frac{\delta}{\lambda\delta+m(\lambda)}\]
\[=\frac{f(k)m(\lambda)-k\delta}{\lambda\delta+m(\lambda)}\]
Taking the first order condition with respect to capital yields:
\[\frac{m(\lambda)f'(k)-\delta}{\lambda\delta+m(\lambda)}=0\]
\[\iff m(\lambda)f'(k)=\delta\]
The first order condition with respect to $\lambda$
\[\frac{m'(\lambda)f(k)\bigl(\lambda\delta+m(\lambda)\bigr)-\bigl(\delta+m'(\lambda)\bigr)\bigl(f(k)m(\lambda)-k\delta\bigr)}{\bigl(\lambda\delta+m(\lambda)\bigr)^2}=0\]
\[\iff m'(\lambda)f(k)\bigl(\lambda\delta+m(\lambda)\bigr)-\bigl(\delta+m'(\lambda)\bigr)\bigl(f(k)m(\lambda)-k\delta\bigr)=0\]
